Fundamentos Teóricos

La digitación es un componente esencial de la técnica pianística que trasciende lo puramente mecánico. Su elección depende del contexto musical, la intención expresiva, la comodidad anatómica y la eficiencia del movimiento. No existen reglas universales ni soluciones únicas; la digitación se construye a partir del análisis y la experiencia del intérprete.

Los datos históricos humanos de digitación hacen referencia a la información contenida en partituras, métodos y tratados de diferentes autores, donde ya viene indicada la digitación para cada pasaje musical. Estos documentos recogen decisiones de digitación tomadas por pianistas y pedagogos a lo largo de la historia.

La generación automática de digitaciones pianísticas ha sido abordada desde diferentes perspectivas. Entre los trabajos más representativos se encuentran las propuestas de Nakamura, Saito y Yoshii [4] (Modelos Ocultos de Markov), Nakamura, Ono y Sagayama [5] (merged-output HMM), y Balliauw, Herremans, Palhazi Cuervo y Sörensen [6] (Variable Neighbourhood Search).

Plan de Evaluación del Modelo

Configuración experimental para el reconocimiento de gestos con sensores

Estructura de Datos

Dedos evaluados: 5 (Pulgar, Índice, Medio, Anular, Meñique)

Variable: DEDOS = [1, 2, 3, 4, 5]

Secuencias por dedo: 6 mediciones temporales

Variable: LONGITUD_SECUENCIA = 6

Objetivo: Evaluar el reconocimiento de gestos para cada dedo

Configuración Experimental

Precisión numérica: ε = 0.001

Variable: EPSILON = 0.001

Proporción de prueba: 20% del total

Variable: TEST_SIZE = 0.20

Semilla aleatoria: 42 (reproducibilidad)

Variable: RANDOM_STATE = 42

Métricas de Evaluación

Exactitud (Accuracy): Porcentaje de aciertos global

Precisión por dedo: Rendimiento individual de cada dedo

Matriz de confusión: Errores entre dedos similares

Metodología: Entrenamiento con el 80% de los datos y evaluación con el 20% restante. Semilla fija (RANDOM_STATE = 42) para garantizar resultados reproducibles. Cada secuencia tendrá 6 mediciones por dedo (LONGITUD_SECUENCIA = 6).

Proyecto de digitación pianística

¿Cómo se alcanzó el objetivo?

1. Comprender la digitación humana

Construcción del conjunto de datos a partir de digitaciones reales.

2. Representar la digitación como un proceso probabilístico

Modelado de las relaciones entre los dedos mediante probabilidades.

3. Incorporar el contexto musical

Consideración de intervalos, dirección melódica y vecindad de notas.

4. Aprender de históricos humanos

Uso de memoria local para recuperar patrones previamente observados.

5. Sugerir digitaciones coherentes

Generación de alternativas de digitación para nuevas secuencias.

El objetivo no fue encontrar una única digitación correcta, sino modelar las relaciones observadas en digitaciones humanas para generar sugerencias coherentes dentro de un contexto musical.
