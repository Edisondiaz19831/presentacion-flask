HMM - Cálculo del prior inicial

Esta etapa calcula la probabilidad inicial de comenzar una secuencia musical con cada dedo.

Idea central:
Para calcular el prior inicial solo se toman los eventos donde is_first = 1. Es decir, únicamente las notas que aparecen al inicio de cada secuencia musical.

Fórmula utilizada

$$\pi_j = P(X_1 = j)$$ $$\pi_j = \frac{\#(X_1 = j) + \varepsilon} {\sum_{r=1}^5 \#(X_1 = r) + 5\varepsilon}$$

Donde πj representa la probabilidad de que una secuencia musical comience usando el dedo j. El término ε corresponde a un pequeño suavizado para evitar probabilidades nulas.
